# Crop Yield Prediction Tool

A Windows Forms application built in C# for importing crop data from Excel and storing it in SQLite.

## Features
- Browse and import Excel files
- Store data in SQLite database
- View imported data in a grid
- Responsive layout with dynamic resizing
- Navigation between forms

## How to run
1. Open `CropYieldPrediction.sln` in Visual Studio
2. Press F5 to run
3. Browse and select an Excel file
4. Click Import to load data into SQLite

## Sample data
Place your Excel file in `Assets/sample_data.xlsx`.
